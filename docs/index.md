AuthKit is a reusable Laravel authentication kit providing:

- Blade UI
- Email verification
- Two-factor authentication
- Password reset
- Rate limiting
- Extensible auth flows

## Features

- Login / Registration
- Email verification (token or link)
- Two-factor authentication
- Password reset
- Configurable auth identity
- Package-friendly architecture
